Etix.com Schema.org Implementation

Focus: Development
Worked with: Java Server Pages, HTML, Schema.org metadata standard

I implemented Schema.org metadata recommendations on the Etix site in order to provide structured data in search results in supported search engines. This made heavy use of the Event, Organization, and Place schemas so that potential ticket buyers could get more useful search results. The accompanying screenshot shows an example of this structured data being used by Google to improve search results.
