IBM Digital Marketplace Search UI

Focus: Development
Worked with: Node.js, React, Sass, Gulp, Mocha, Webpack, TravisCI, Bluemix (PaaS)

Development

As part of the same team maintaining the IBM Digital Marketplace storefront, I wrote UI components and unit tests for the search results pages of the Digital Marketplace.

This application had its own CI/CD pipeline setup that we maintained, consisting of GitHub Enterprise, TravisCI, and Bluemix. Search results were rendered from an Elastic Search endpoint that another team maintained as a microservice.
